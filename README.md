# Mohamed Amine Gotai — Portfolio Data Science

Portfolio single-page prêt pour GitHub Pages.

## Modifier le contenu

- **Informations personnelles** : éditez `index.html` (sections Hero et Contact).
- **Projets** : dans la section `#projects`, remplacez chaque carte placeholder :
  - `title` (titre)
  - `summary` (résumé)
  - `tools` (technologies)
  - liens `GitHub` / `Demo`
- **Compétences / Expérience / Certifications** : mettez à jour les sections `#skills`, `#experience`, `#certifications`.

## Ajouter le CV

1. Placez votre fichier au chemin exact : `/cv.pdf` (racine du dépôt).
2. Le bouton **Download CV** est déjà configuré.
3. Si le fichier est absent, une note s'affiche automatiquement et le bouton est désactivé.

## Personnaliser le style

- Le design est dans `styles.css` (variables CSS en haut du fichier).
- Le comportement interactif est dans `script.js` (menu mobile, année auto, lien actif, vérification CV).

## Activer GitHub Pages

1. Ouvrez **Settings** du dépôt GitHub.
2. Allez dans **Pages**.
3. Dans **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Sauvegardez puis attendez le déploiement.

Votre site sera disponible sur : `https://aminegottai.github.io/`
